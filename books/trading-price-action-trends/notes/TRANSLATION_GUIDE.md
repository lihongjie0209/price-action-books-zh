# 翻译规范（供译者/代理使用）

## 目标
将 `en/chapters/*.md` 译为简体中文，输出到 `zh/chapters/` **同名文件**。

## 必须遵守
1. 阅读 `glossary/glossary.md`，术语统一用「首选译名」。
2. 保留 Markdown 结构：标题级别、列表、`<!-- PDF page N -->` 注释、水平线。
3. 正文为通顺简体中文完整句子，禁止机翻碎句、中英混杂正文（专名除外）。
4. 人名、软件名、合约代码、URL 保留英文。
5. High 1/2、Low 1/2、Always-In 等 Brooks 编号/模式名可保留英文并括注中文。
6. 图表说明（Figure x.x）译为「图 x.x」，说明文字译中文。
7. 不要删减原意；不要添加译者评注（除非原书脚注）。
8. 文件开头保留章节标题（中文），可在标题下用 HTML 注释保留英文原标题。
9. 输出必须是**完整章节**，不能只译前几段。

## 质量标准
- 与 EN 源文件篇幅同量级（中文汉字数通常约为英文词数的 0.6–1.2 倍字符量级，禁止标题-only 空壳）。
- 同一术语全章一致。
